# PCA algorithm

This repository completes the task of the course **Machine Learning**, which requires to eliminate the noise of a dataset via PCA algorithm, so that we can analyse and tackle the data in it.

## About the report 
You just need to open `tex/main.pdf`, and you can see the report. To see more, please enter `tex/main.tex` to vie the latex code.

## Running Procedure
First install the required packages
``` bash
pip install -r requirements.txt
```
Then run the code
```bash
python main.py
```
